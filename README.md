# Softball-HallofFame-Website

## Project Overview
This project involved contributing to the modification and improvement of an existing WordPress-based website for the Columbus, Indiana Slow Pitch Softball Hall of Fame. The goal was to enhance the siteâ€™s structure, content presentation, and usability while preserving its historical and community-focused purpose. The website highlights Hall of Fame members, historical information, and community involvement related to slow pitch softball in Columbus, Indiana.

## Tech Stack
- WordPress
- WordPress Themes
- WordPress Plugins
- Web Hosting (BlueHost)

## My Role
I contributed to modifying and improving the existing WordPress website. My responsibilities included updating page layouts, adjusting content structure, configuring plugins, and making visual and usability improvements to ensure the site was clear, accessible, and easy to navigate.

## Key Contributions
- Modified existing WordPress pages and layouts
- Updated content related to Hall of Fame members and history
- Improved navigation and page organization
- Adjusted theme settings to enhance visual consistency
- Ensured responsiveness across desktop and mobile devices
